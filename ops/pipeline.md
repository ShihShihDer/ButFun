# Pipeline / 案件看板

> 一人公司的 CRM。每個 lead / project 在這張表上有一筆。
> 詳細評估在 `leads/<代號>.md`,執行紀錄在 `projects/<代號>/`。

---

## 圖例

| Stage | 意義 |
|---|---|
| `INBOX` | 剛貼進來,還沒評估 |
| `EVALUATED` | 已評估,等你決定要不要投 |
| `BIDDED` | 已投 proposal,等回覆 |
| `NEGOTIATING` | 客戶有回,在談細節 / 報價 |
| `WON` | 接到了,等簽約 / 訂金 |
| `IN_PROGRESS` | 開發中 |
| `DELIVERED` | 已交付,等客戶驗收 |
| `PAID` | 收款完成 ✅ |
| `LOST` | 沒接到 (記錄原因,優化下次) |
