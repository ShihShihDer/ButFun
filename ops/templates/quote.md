# 報價單模板 / Quote Template

> 給客戶看的版本 — 三段式 (Premium / Standard / Lite),引導選 Standard。
> 內部估時記錄在 `leads/案件代號.md`,不要給客戶看。

---

## 中文版

```
[客戶稱呼] 您好,

根據前次討論的需求,以下是我的提案。

────────────────────────────────────
方案 A — Premium    NT$ [X * 1.5]
────────────────────────────────────
✓ 完整功能交付
✓ 完整 API / 部署文件
✓ 自動化測試覆蓋
✓ 1 個月免費 bug fix 保固
✓ 優先回覆 (4 hr 內)
✓ 上線後 1 次架構諮詢

────────────────────────────────────
方案 B — Standard   NT$ [X]   ⭐ 推薦
────────────────────────────────────
✓ 完整功能交付
✓ 部署文件
✓ 核心功能測試
✓ 2 週 bug fix 保固

────────────────────────────────────
方案 C — Lite       NT$ [X * 0.7]
────────────────────────────────────
✓ 核心功能交付
✗ 無文件
✗ 無測試
✗ 無保固

────────────────────────────────────

付款方式 (擇一):
- 預期工期 < 1 週: 100% 預付
- 預期工期 1-3 週: 50% 簽約 / 50% 交付
- 預期工期 > 3 週: 拆 milestone,分 3-4 期

預計交付時程: [N] 個工作天 (從收到訂金 + 完整需求文件起算)

報價有效期: 7 天

附註:
- 規格變更需另計
- 不含第三方服務費用 (API token / 雲端主機等)
- 來源碼智財權於尾款收訖後完整移轉

期待您的回覆。

— [你的名字]
[聯絡方式]
[日期]
```

────────────────────────────────────
✓ Full feature delivery
✓ Complete API + deployment docs
✓ Automated test coverage
✓ 1-month bug-fix warranty

────────────────────────────────────
Option B — Standard     USD [X]    ⭐ Recommended
────────────────────────────────────
✓ Full feature delivery
✓ Deployment guide
✓ Core feature tests
✓ 2-week bug-fix warranty

────────────────────────────────────
Option C — Lite         USD [X * 0.7]
────────────────────────────────────
✓ Core feature delivery
✗ No documentation
✗ No tests
✗ No warranty

────────────────────────────────────

Payment terms (one of):
- < 1 week scope: 100% upfront
- 1-3 week scope: 50% on signing / 50% on delivery
- > 3 week scope: 3-4 milestones with independent acceptance

Estimated delivery: [N] business days from deposit + finalized spec.

Quote valid for: 7 days.

Notes:
- Scope changes require re-quoting
- Third-party costs not included (API tokens, hosting, etc.)

Looking forward to your reply.

— [Your Name]
[Contact]
[Date]
