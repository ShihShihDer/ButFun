# 接案平台清單 / Platform Catalog

> 按優先順序排列。優先級評估標準: **單位時間期望收入 = 平均時薪 × 命中率 ÷ 投標成本**
>
> 你的定位 (AI 整合 + 全端 + DevOps) 在英文高價平台 ROI 最高,中文平台則靠速度與品質取勝。

---

## Tier S — 最高優先 (先攻這幾個)

### 1. Upwork (英文,全球)
- **URL**: https://www.upwork.com
- **優勢**: 案量最大、AI 整合需求爆炸、付款機制成熟
- **劣勢**: 競爭激烈、要花 Connects 投標、抽 10%
- **適合你的關鍵字**: `Claude API`, `Anthropic`, `LLM integration`, `AI agent`, `prompt engineering`, `Spring Boot`, `Kubernetes`, `CI/CD`, `iOS Swift`
- **註冊建議**: profile 100% 完成度、選 3-5 個 skills 不要超過 10 個、上 1-2 個 portfolio item
- **首月策略**: 投 30 個,目標命中 1-2 個小案累積 reviews

### 2. Contra (英文)
- **URL**: https://contra.com
- **優勢**: **0% 手續費** (Upwork 抽 10%)、適合 independent
- **劣勢**: 案量比 Upwork 少
- **適合**: 已經有點 portfolio 的人

### 3. 104 外包網 (中文,台灣)
- **URL**: https://case.104.com.tw
- **優勢**: 台灣最大、付款保障、客戶多為中小企業
- **劣勢**: 報價普遍偏低、會殺價
- **策略**: 主動投案 + 把 profile 做好讓客戶找你

### 4. Workana (西文/葡文,拉美)
- **URL**: https://www.workana.com
- **優勢**: 拉美最大、西文競爭比英文平台少、北美客戶會來找成本較低的
- **劣勢**: 案件預算不如 Upwork
- **策略**: 西文 proposal 要在地化(用 Claude 確保不是 google 翻譯腔)

## Tier A — 第二波 (拿到首單後再開)

### 5. Toptal (英文)
- **URL**: https://www.toptal.com
- **優勢**: 號稱前 3%,案件單價最高 (USD 80-150/hr)
- **劣勢**: 要過嚴格 screening (英文面試 + coding test + trial project)
- **時機**: Upwork 累積 5-10 個 5 星評價後再申請

### 6. Malt (歐洲多語,含西文)
- **URL**: https://www.malt.com
- **優勢**: 歐洲企業客戶、預算高、可用西文
- **劣勢**: 主要服務歐洲時區

### 7. Fiverr (英文,productized)
- **URL**: https://www.fiverr.com
- **策略**: 不靠投標,靠包套餐 — 例如「我幫你建一個 Claude API 整合的客服 bot,USD 500 起」
- **適合你的 gig 點子**:
  - "I will integrate Claude API into your app/website"
  - "I will set up CI/CD pipeline with Jenkins / Kubernetes"
  - "I will build an AI agent workflow with Claude Code"

### 8. CakeResume 接案 (中文,台灣)
- **URL**: https://www.cakeresume.com
- **優勢**: 新創/科技業客戶較多、客單價較 104 高
- **策略**: profile 用英文 + 中文雙語

### 9. Freelancer.com (英文,全球)
- **URL**: https://www.freelancer.com
- **劣勢**: 印度/孟加拉 cut-throat 競價、平均品質差
- **策略**: 只接客戶有付費 verified 的案

## 監控策略

> 我沒辦法登入這些平台幫你掃案子。但你可以做的事:
>
> 1. 每天 10-15 分鐘,登入 Upwork / Workana / 104 各看 5 分鐘
> 2. 把符合關鍵字的案件**整段複製**貼到 `inbox/YYYY-MM-DD-平台-關鍵字.md`
> 3. 跟我說「評估 inbox 的所有案件」,我會在 `leads/` 產出評估報告 + proposal 草稿
> 4. 你只負責決定要投哪個、把 proposal 貼出去

### 進階自動化 (有興趣再做)
- 用 RSS / email alert 訂閱平台關鍵字推送
- 寫 Chrome Extension 一鍵把案件 dump 到 `inbox/`
- 部署一個 Telegram / LINE bot,你拍照案件 → 自動進系統
