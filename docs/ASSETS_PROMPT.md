## 世界

ButFun 是一款 2D 俯視瀏覽器即玩的多人線上世界,主題:**蒸汽龐克太空歌劇 + 療癒拓荒**。
背景設定在「大靜默」之後——乙太(整個星系的魔法燃料)曾突然崩潰、文明殞落,
如今乙太緩慢回流,玩家是回到邊境星的拓荒者,從繼承的乙太工坊與農莊開始重建,
終極走向多星球星際。

## 視覺基調

- **像素風(pixel art)**,每格 **32×32 px**,2D 俯視(top-down)
- 色票:**暖土黃 + 黃銅(brass)+ 苔綠 + 深星空藍**,飽和度中等
- 風格:**冷色基底 + 暖色飾邊**(蒸汽龐克的黃銅金 + 太空的鈷藍)
- 氣質:**久居、溫暖、有人氣的舊**(不要科幻冷感、不要迪士尼乾淨感)
- 夜晚會疊暗藍濾色,所以日色不要太低明度

## 統一色票(請維持同批一致)
- 草地深綠 `#2d4d2a`
- 草地亮綠 `#3e6a3a`
- 泥土暖黃 `#7a5f3c`
- 翻土深褐 `#5b4636`
- 黃銅金 `#c9a24b`
- 文字米白 `#e8e0cf`
- 深空底色 `#0b0d12`

## 這批要做的(優先順序由上往下)

### 1. 地面 tileset(最急)
單張 PNG,4 列 × 4 欄,每格 32×32,**無縫可貼**(平鋪在大面積不接縫):
- 第 1 列:草地 4 變體(同色微差,避免大面積機械重複)
- 第 2 列:泥土路徑 4 變體
- 第 3 列:水面 4 影格動畫(微波)
- 第 4 列:石頭/石板 4 變體(工坊周圍用)

### 2. 田地系統 sprite(緊隨地面)
單張 PNG,一列 8 格,32×32:
- 未翻土(略乾自然地,跟草地能區分)
- 已翻土(深褐有犁溝)
- 已澆水(深褐泛濕反光)
- 乙太作物階段 1:種子(小土點)
- 乙太作物階段 2:發芽(綠莖+兩片小葉)
- 乙太作物階段 3:成長(更高、有小花苞)
- 乙太作物階段 4:成熟(發光金果——主角元素,要有「乙太魔法燃料」的金黃光芒)
- 乾燥提示(可選):藍色點點 overlay

### 3. 木柵欄(田地周圍)
單張 PNG,32×32 每格:水平、垂直、左上角、右上角、丁字、十字。
**蒸汽龐克味**:不只是木頭,可加黃銅鉚釘飾邊。

### 4. 玩家角色 placeholder(之後會做完整創角)
32×32 每格,單張 PNG sprite sheet:**4 列(下/左/右/上)× 4 欄(走路 4 影格)**。
中性人型,**戴蒸汽龐克護目鏡 + 圍巾**(辨識度),色調黃銅+米白。
> 這只是 placeholder——遊戲計畫做**全自由角色創造器**,身體部位會分開上色,
> 所以這張先用同色塊體型就好,之後會替換成程式組裝的角色。

### 5. UI 圖示(若還有額度)
16×16,單色線條 + 透明背景:`💡 建議` / `🌱 田地` / `⚙️ 設定` / `📦 背包` / `🚗 載具`

## 輸出格式約束

- 每張 PNG **透明背景**
- 同檔內各 sprite 之間 **0px gap**(緊貼,我們用程式硬讀)
- 解析度就是 **32×32 一格**,**不要放大插值**(會糊掉像素邊)
- 授權:**CC0 / Public Domain 或可商用**(會放進開源 repo)
- 同一批內**色票嚴格一致**(用上方的 hex)
