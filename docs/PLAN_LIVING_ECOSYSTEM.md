# 活生態系 — 從「人類 vs 野怪」到一個會自己活的世界

> 願景來源：**玩家 2026-06-12**——「人類 vs 野怪太局限，應該是個生態系：有食物鏈（野貓野狗吃野鳥、
> 有強弱階級）、死亡會回歸乙太或滋養食腐動物、各物種有自己的城鎮與自保能力、人類跟各物種可能打仗
> 也可能互助」。模型/拆解：AI 規劃。延伸 `VISION_AI_EMERGENT_ECOSYSTEM.md`。

## 核心轉向：野怪 → 物種

把「野怪＝敵人」這個框拆掉。世界上是**各種物種/生靈**，**敵對只是牠們可能的關係之一，不是定義**。
- 有的會獵食、有的被獵食、有的中立路過、有的可結盟。
- 牠們各有**需求**（食物 / 安全 / 領地 / 繁衍），行為從需求**湧現**——不寫死「貓恨鳥」，而是給「貓要吃、鳥是食物」，
  獵食、逃跑、群聚、遷徙自然長出來。

## 三條生態支柱

1. **食物鏈 / 獵食**：物種分營養階（草食 → 小型肉食 → 大型肉食 / 菁英）。捕食者追獵物、獵物逃跑群聚；
   族群數量隨食物多寡起落（獵物多→捕食者增；獵物少→捕食者餓→減）。湧現的平衡，不是固定刷怪。
2. **死亡餵養生命**：死亡不是憑空消失——**回歸乙太**（平和、釋放乙太微粒）**或**留下屍體**滋養食腐者 / 土壤
   （長出資源 / 餵下一環）。死亡是循環的一環，不是終點。
3. **物種聚落 + 自保**：各物種可有自己的**小聚落 / 巢穴 / 城鎮**，有**自保能力**（守衛、陷阱、群體防禦）。
   世界上散布各式各樣的聚落，不只人類城。

## 人類 ↔ 物種：湧現的關係（不寫死敵我）

接「湧現派系（ROADMAP 71）」那套到跨物種：人類與各物種之間有**好惡 / 信任**，從互動長出來——
- 可能**打仗**（搶領地 / 獵食威脅 / 報復），也可能**互助結盟**（互惠貿易、共禦更大的威脅、馴養共生）。
- 玩家的行為（獵殺 vs 餵食 vs 保護）影響整個物種對人類的態度。**分久必合、合久必分**，跨物種也適用。

## 漸進建造順序（一次一塊；先救眼前、再長生態）

> 鐵律：**湧現不寫死**（給需求/食性，讓行為長出來）、**成本紀律**（生態模擬用便宜啟發式，LLM 只留給
> 「有智慧的物種領袖」且重限流）、**效能**（大量生物要省）、原創美術、資料安全、一次一塊、走 review。

1. **(139) 圍城平衡——先把獸潮壓回合理**（眼前最痛，且它正壓死城鎮繁榮→害居民不長）。
   降低頻率/規模或**隨城鎮防禦縮放**；確認 114（無人也跑）+ 117（怪物王指揮）沒疊加成「永久圍爆」；
   讓玩家打得退、城鎮喘得過氣（安全感回升→繁榮→居民才會增）。順手：居民起始數略高 / 視覺更明顯，讓玩家看得到。
2. **(140) 物種化 + 中立野生動物**——導入**不敵對**的生靈（野鳥/野鹿/小動物），只是生活在世界裡；
   把「野怪」概念正名為「物種」，敵對改成「關係之一」。世界第一次有「不是來打你的」生命。
3. **(141) 食物鏈 / 獵食**——捕食者獵食、獵物逃跑群聚；族群數量隨食物起落（湧現平衡）。
4. **(142) 死亡餵養生命**——死亡→回歸乙太（釋乙太）或留屍滋養食腐者/長資源。
5. **(143) 物種聚落 + 自保**——各物種有巢穴/聚落與群體防禦；世界散布多樣聚落。
6. **(144) 人類↔物種湧現關係**——跨物種好惡/信任，打仗或結盟由互動長出（接 ROADMAP 71）。

## 為什麼這是靈魂

現在的世界是「人類城 + 來打你的野怪」。做完這條，世界變成**一個會自己活的生態系**——
你掛 0 人放著，回來會看到：族群此消彼長、有物種興起有物種式微、某個聚落跟人類結了盟、某條食物鏈崩了又重組。
這跟「自走世界（114~116）」「真實經濟（100~108）」合起來，就是「**一個不需要你也持續演化的活世界**」。
