## 接下來的切片(薄切片快速連發,由上往下)

1. **薄採集→背包:走近採到、進背包、看得見**(玩法鏈主幹,backend 主力)
   驗收 = 玩家走到曠野節點旁按鍵/點一下 → 木/石/乙太進可開關「背包」面板 → 跨重連還在。
   只做這最小迴圈,**別**一次把合成/裝備/工具效用全做進來。
2. **改名薄切片**(高優先、並行):設定面板加改名輸入框 → 後端 `PATCH /api/profile`(過
   `sanitize_name`)→ 持久化 → 即時反映 HUD/聊天/快照。
   驗收 = 玩家改名後自己與他人即時看到、重連仍是新名。
3. **物品有用途**:採到的東西能幹嘛(賣乙太 / 餵作物 / 當材料),讓背包不只是清單。
4. **戰鬥 MVP(自動打怪)**:世界有可打目標、角色自動攻擊附近敵人、掉落進背包。先最薄一條。
5. **(並行,別擋路)持久化**:背包/位置/農地狀態存 Postgres——能先上就先上、隨後補。
