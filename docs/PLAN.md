## 🎯 現在主攻:合成(1-C)——給暴增的背包素材一個「去處」,玩法鏈裡缺的那一環

玩法鏈已接通三環:**採集→背包(PR#18)、打怪掉落→背包(戰鬥 1-F,PR#21)、農地收成→乙太(0-G)**。
戰鬥剛上線(自動鎖定/掉落進背包/生命值/回血/無障礙報讀都齊),解掉了「世界太靜/想要會動的東西」。

**但這同時把老痛點放大了**:採集、打怪、農地**三個來源**現在都在灌背包(木/石/廢料/乙太),
玩家手上素材堆得比以前更快——**卻幾乎沒有任何去處**。乙太只進不出、木石純囤積。
GDD 紀律白紙黑字「乙太有產出也要有去處」此刻是**最大缺口**,比再加新產出來源都重要。

對應兩個去處,照玩法鏈往下連發:
- **素材的去處 = 合成**(背包木/石/廢料 → 工具)。`crafting.rs` **已落進 main(commit 1a91799)**,
  但只是 `mod crafting;` 宣告、`ws.rs`/`game.rs` 沒接 `Craft`——**檔已就位、只差接線,別再去 feature lane 找**。
  且 **additive、不動廣播 shape**:client 送 `Craft` → 伺服器查配方扣素材 → 背包走**既有**快照,
  前端只多一個面板 = **零契約變更**,本 repo 最好 land 的那種 → **本輪第一優先**。
- **乙太的去處 = 擴地**(收成換更多地)。`economy.rs` 已建好沒接,排近期。

### 給維護者的接線優先序(一次一條,別並行撞 main)
1. **先 land 合成(1-C)**:`crafting.rs` 已在 main(配方已建、additive 同採集接法),只差在
   `ws.rs` 收 `Craft`、`game.rs` 查配方扣素材加道具。背包已隨快照廣播,合成不動契約,風險最低、最快見效。
2. **再 land 工具用途(1-D)**:身上有鎬子採礦更快(`tools.rs` 已建)。閉合「素材→合成→更快採」
   第一個完整正回饋圈,讓合成出來的東西真的有用。
3. **擴地(0-G-O2,乙太買地)**:摘 `economy.rs`。per-player fields 已是 Vec 廣播,能複用既有 shape
   就低風險先上(同玩家認領相鄰新地塊);需動 shape 才走 PR。

## 接下來 1-3 個切片(由上往下,每個附「驗收=玩家能做到什麼」)

1. **合成 MVP(1-C)— 最低風險、先 land**:木×3+石×2 → 鎬子,不動廣播 shape。
   **驗收 = 玩家把背包素材(採集+打怪掉落)合成出鎬子、囤積終於有去處**(經濟迴圈起步)。
2. **工具效用(1-D)— 閉環**:有鎬子採礦更快、沒有就慢。先只做鎬子×採礦一條。
   **驗收 = 合成出的鎬子真的讓採集變快**,第一個完整正回饋圈。
3. **擴地 MVP(0-G-O2)— 乙太的去處**:收成/打怪的乙太換一塊相鄰新地塊。薄:固定價、固定下一格、無市場。
   **驗收 = 玩家花乙太多拿一塊地、填滿後不再停滯**。
