## 🎯 現在主攻:戰鬥 / 自動打怪 接線(1-F)

採集→背包已上線(PR #18),但**背包目前是死路**——採到的東西沒地方用、世界裡也沒有會動的東西。
繼採集後**最大的可玩鉤子=讓世界活起來**:有敵人在晃、走近自動打、掉落直接進背包。
零件現已 **100% 齊全且去風險**:`auto/feature` 的 `combat.rs`(敵人/扣血/掉落/重生)、`enemy_field.rs`
(世界佈置+走近自動鎖定)、`vitals.rs`(玩家生命/被打趴休息復原)、威脅聚合,**還附了跨模組組合測試鎖住接縫**。
**照採集→背包同法接即可**,純接線、不必造任何零件。
> 為什麼這個最提升可玩性:它一次給「新活動 + 背包有產出 + 世界有生氣」三件事,且零件齊+測試在,接線最快。

  全建好沒接,再加只是囤貨。**只做能獨立 land 的小事**:敵人**資料**(種類/血量/掉落表/出沒點)、

- **frontend**:做能獨立 land 的小事——**敵人 sprite / 血條 / 受擊飄字的畫法**(本輪主攻會用到,先備好)、
  背包面板打磨。別再堆農地微打磨。分支卡衝突就從 main 重開乾淨枝。
- **backend**:小而獨立——改名 `PATCH /api/profile` 收尾、持久化補平(背包/節點存 Postgres)。
  大接線(戰鬥/合成進 ws.rs)由**維護者一次一條**做,別搶。
