## 「主軸 vs 補洞」判準（worker 與 reviewer 都照這個）
- 會讓玩家**看到新東西 / 新玩法 / 更大的世界** → 主軸，做。
- 只是讓既有東西**更安全 / 更快 / 更乾淨、玩家無感** → 補洞，**除非擋路否則跳過**。
- 玩家建議（`data/suggestions.jsonl`）：符合主軸的排進來；要取捨的升級給人（`for_human.md`）。
